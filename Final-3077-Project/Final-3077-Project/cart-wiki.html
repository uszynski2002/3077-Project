Wiki: How to Edit Product page

Step 1

Locate the product.json file

In MyWeb, the file will be in the 2707-project folder

Step 2

Find Item to Edit

Using Hot Chocolate as an example,
create a new name for the item.
This will cause the page to disconnect

Step 3

Reconnect JSON info to page

Open the HTML for Hot Chocolate (product3.html).
There will be a data attribute in the body element.
(It is labelled in the comments UPDATE ONLY THIS)
Replace the current title with the newly assigned one,
make sure to include caps if used

Step 4

Finish Updating JSON

Now head back to product information in JSON
Replace all parts.
When adding a new image, make sure to copy file into the image folder within the 2077-project folder.

The new product has now successfully been updated on the website! including the front page and the Shopping Cart
